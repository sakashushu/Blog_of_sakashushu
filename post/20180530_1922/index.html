油断すると思考停止

2018/05/30 2 分で読める

知ってる人は知ってますが、 2015年10月末、ボクは1年以上続く無職生活に入りました 。

その時に強く意識して、今後も気を付けようと思ったことに、

思考停止しない

ということがあります。

でもこれがナカナカ、油断するとすぐにやっちゃってます。

　

昨夜も、中１の長男が学校から「アルトリコーダー購入申込書」を持って帰ってきました。

その瞬間はボクは思考停止していて、封筒にお金を入れて渡しました。

でも、自分でそのこと（思考停止していたこと）に気付いて、可能性の模索を始めました。

そして、今日のボクの動き。

ご近所さん（２軒お隣の高校生がいるお宅）にヒアリング

→ 残念ながら当時の音楽の先生はリコーダーでなく篠笛を教えていたので、リコーダーは不所持

→ 実家に連絡し、ボクの中学時代のアルトリコーダーが無いかヒアリング

→ 母親が探してくれて、夜に何とか発見！

→ 学校からのプリントには、バロック式でないとダメとある

→ ネットでバロック式の特徴を調べる

→ 母親にその特徴を伝えたところ、バロック式であることが判明！

って訳で、ボクが中学の頃（３０年前）に使っていたアルトリコーダーを取りに行く流れになりました（笑）

　

本来（思考停止していない時に）ボクが大事にしているのは、金額だけで物事を考えないことです。

大した金額じゃない

とか、

新品買った方が安い

とかは、

お金の奴隷になっている側面があると思ってます。

実家にリコーダーを取りに行くのだって、

ガソリン代を考えたら買った方がまし

って捉え方もあるとは思います。

でも、ボクは子供のようにシンプルに考えたいです。

そこにまだまだ使えるアルトリコーダーがあるなら、それを使うでしょ（｀・ω・´）ゝ キリッ

※思考停止してて、危うく買っちゃうトコだったけどね ( ￣3￣)～♪
